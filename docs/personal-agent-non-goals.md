# Personal Agent Non-Goals

Angella는 개인용 frontier-first agent harness입니다. 아래는 의도적으로 기본 경로에서 제외하는 항목입니다.

## Non-Goals

- enterprise observability stack
- commercial SLA or enterprise-grade support posture
- global auto-install of experimental tooling such as `qmd`
- global auto-hook rewriting of every shell command
- personal PKM vault auto-ingest by default
- local-only default operation

## Optional Experimental Slots

- `qmd`
  - optional search provider
  - never a mandatory dependency
